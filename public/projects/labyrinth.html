Labyrinth 

A Maze game created using Vanilla Javascript

Used Randomized depth-first search algorithm and Recursive Backtracking technique for Maze creation.

An initial cell is chosen, marked as visited and pushed to the stack. If the current cell has any neighbours which has not been yet visited then the current cell is pushed into the stack. Any of the unvisited neighbour is chosen & the wall between the current cell and the chosen cell is removed. The chosen cell is marked as visited and pushed into the stack.

A step tracker and a time tracker is also implemented to mark down the player's statistics.

As the player completes the puzzle, a greeting prompt is thrown which shows the round statistics and allows if player wants to restart.
