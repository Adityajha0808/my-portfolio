My Memo 

A To-do list App created using React library

Built using useState() hook of React; Passing props through components, handling events and rendering elements.

A task with a title is typed into the input field and submitted. Once submitted the note will appear as an entry. We can keep adding items to add additional entries in card format and have them all show up. Apart from entry addition, their is also a deletion icon in the note which removes the specific task and lists rest of the tasks accordingly.

The parent component is App, which stores the notes array in its own state, managed using the useState() Hook. Inside App component, we create a list of those components, which will be passed by Note component from the notes prop using createArea component. The createArea component contains handleChange function which tracks the data from the user end and passes it to App.

The createArea component iterates using map() function and renders a list of notes.
